# Tauri + SvelteKit + TypeScript

This template should help get you started developing with Tauri, SvelteKit and TypeScript in Vite.

## Get started

- `cd phenoxtract-ui`
- `npm install -g pnpm`
- `pnpm install tailwindcss @tailwindcss/vite`
- `pnpm tauri dev`

## Formatting

Open RustRover → Settings → Editor → Code Style, then switch tabs to Formatter and add
`*.html *.ts *.js *css *svelte` to `Do not format`.

Next open plugins and install the Biome plugin. Go to Settings → Languages & Frameworks → Biome and check
`Enabel LSP-based Formatting`
